# Tea Range Images

Place your tea category images in this folder.

## Required Images:

1. **freshness.jpg** - Image for Freshness category
2. **bubble-tea.jpg** - Image for Bubble Tea category
3. **flavorful.jpg** - Image for Flavorful category
4. **fruit-tea.jpg** - Image for Fruit Tea category
5. **sustainability.jpg** - Image for Sustainability category

## Image Specifications:

- **Recommended size**: 600x400px or similar aspect ratio
- **Format**: JPG, PNG, or WebP
- **Aspect ratio**: 3:2 or 4:3 works well

## Notes:

The images will be displayed in a grid layout with text overlay showing the category name.
Images will zoom slightly on hover for a nice interactive effect.
